# Atelier Narkis · סידורי מהדורה

Static 3-page luxury micro-site for a limited-edition Hebrew siddur
signed by the Israeli singer Narkis.

**Pages**
- `index.html` — signature / home
- `siddur.html` — the product, with live dedication preview
- `story.html` — narrative + reservation form

**Stack**
- Vanilla HTML + CSS + JS (no build step)
- Hebrew RTL, `Frank Ruhl Libre` / `Cormorant Garamond`
- Black & gold luxury design system

**Deploy**
Any static host. Configured for Vercel via `vercel.json`.
